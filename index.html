Projeto Frontend

Utilizando a API Kitsu, gostaríamos de renderizar uma lista de personagens em formato de cards, conforme o exemplo abaixo.

Nosso designer e nosso desenvolvedor sairam de férias, precisamos melhorar a interface e usabilidade desta página também.
Sinta-se livre para utilizar vanilla javascript ou algum framework, css, sass ou a ferramenta que mais te agradar.

  • Instruções:
  • Renderizar no mínimo 4 cards de personagens
  • Cada card deve conter as seguintes informações do personagem: imagem, nome, breve descrição e outros nomes que ele utiliza.
  • Você pode acessar as informações através da API Kitsu: https://kitsu.docs.apiary.io/
  • O layout da página deve ser responsivo.
  • A entrega do projeto deverá ser feita pelo GitHub. Quando estiver finalizado, nos envie o link de acesso.
  • Lembre-se! Você tem 3 dias para entregar o projeto.
